# Flutter Sembast Task CRUD

A Simple Task App using SEMBAST project to store local data.

### Dependencies
- [path_provider 1.6.7] (https://pub.dev/packages/path_provider)
- [sembast 2.4.3] (https://pub.dev/packages/sembast)
